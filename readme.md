Features:
* Multiview/app render engine
* Applauncher/switcher
* sub-menu's
* Per-view configurable functions & actions.
* Notifications
* Can be used fully local and standalone
* Plug-&-play integration with Home Assistant through ESPHOME or a MQTT Broker.
